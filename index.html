🚀 一、快速入手

1. 下载与安装

Q: 整合包在哪里下载？怎么安装？

A: 群文件内有两个版本：

  • 你好新蒸程1.0.0全量客户端.zip → 解压即玩，适合新手。
  • 你好，新蒸程！正式版 V1.0.0.zip → 需要拖入 PCL2 / HMCL 启动器自动安装。

⚠️ 必须安装 Java 21，启动器会提示，或使用群文件内的 jdk-21_windows-x64_bin.exe。

2. 必备环境与内存

本整合包非常吃配置！推荐内存分配 12GB - 16GB。
启动器设置: PCL2 → 设置 → 启动器 → 游戏特定设置 → 最大内存/内存。

⚠️ 警告：如果你只有8GB物理内存，请勿尝试！至少需要16GB物理内存才能流畅体验VOXY。

🌄 二、性能优化与地形加载 (VOXY)

🌟 推荐：直接使用“跑好的地图”

自己用 /chunky radius 2000 指令预加载地图需要数小时，且极易崩溃！
解决方案：下载群文件中的 贝列瑞安德的原野.zip 或 贝列瑞安德的原野（小）.zip，解压到你的 .minecraft/saves 文件夹即可。

VOXY 闪烁/崩溃/不生效

常见原因及解决：

  • 显卡驱动过旧：更新或回退驱动版本。
  • 内存分配不足：确保给了至少12GB。
  • Intel核显/I卡兼容问题：VOXY可能无法正常工作，建议关闭VOXY或删除 voxy-*.jar。

帧数很低，只有20-30帧

首先，关掉光影。其次使用群内已加载好的地图。如果还卡，在视频设置里降低 渲染距离 到12以下，关闭 垂直同步。

🐞 三、已知Bug & 崩溃修复

❌ 经验颗粒 / 附魔工业 按W思索崩溃

这是 Create Enchantment Industry 模组的已知问题。千万不要按W思索经验颗粒。等待后续整合包更新。可暂时删除该模组，但会丢失相关玩法。

❌ 扳手 + 连锁采集 导致方块消失

这是已知Bug，制作组已提供临时解决方案：安装模组 Create Ultimine 并移除冲突的采集键绑定。后续版本将修复。

❌ 传送石碑 / 抱起飞车 导致视角卡死或崩溃

Carry On 模组与载具交互时易崩溃，已知问题，正在修复。临时解决方法：避免使用该交互，或用撬棍回收载具。

🔧 其他崩溃提示

遇到崩溃时，请将 错误报告（latest.log或crash-report） 发给AI分析（如DeepSeek、Claude），往往能得到最直接的解决线索。

🛠️ 四、游戏玩法 & 指令

🚗 开局送的车怎么收起来/坏了怎么办？

用 撬棍（两个铁锭合成）。对着车 Shift + 右键 即可将其收为道具。如果翻到沟里，也是用撬棍回收。

🔌 为什么无线红石/遥控器在船上无效？

在物理化结构上，请使用 无线打字机 代替遥控器，或在配置文件里修改设定。

📦 简易存储 / 终端面板在哪？

模组改名为 Tom's Simple Storage，配方和以前略有不同，请用JEI查询合成表。

👗 工程师护目镜不能装备到饰品栏？

这是个Bug，可能和饰品模组（Curios API）不兼容，等待修复。目前只能放在普通头盔位。

🌐 五、服务端与联机

服务端文件

群文件已有：新蒸程1.0.0服务端.zip。
配置要求：至少4核 4.5GHz+ CPU，分配12GB+ 内存。
注意：服务端不自带地图，需自行上传跑好的存档。VOXY是客户端模组，服务器无需配置，但玩家本地需要已加载的缓存。

我跟朋友局域网联机，朋友也要跑图吗？

不需要跑图！主机跑好图后，把本地存档里的 .voxy 隐藏文件夹发给朋友，覆盖到他电脑的对应存档内即可继承超视距渲染，详情B站搜索VOXY使用教程。

正版验证失败 / 无效会话

服务器开启了正版验证。你需要在启动器登录正版账号，或让服主在 server.properties 中设置 online-mode=false（但请注意风险）。
